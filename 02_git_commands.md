1. Use "git --version"

To Check Git is Installed or Not

--------------------------------------------------

2. Use "git config --global user.name "Your Name""

To Set Your Git Username

--------------------------------------------------

3. Use "git config --global user.email "your@email.com""

To Set Your Git Email

--------------------------------------------------

4. Use "git config --list"

To View All Git Settings

--------------------------------------------------

5. Use "git init"

To Initialize a New Git Repository

--------------------------------------------------

6. Use "git status"

To Check File Status (Tracked / Untracked)

--------------------------------------------------

7. Use "git add file_name"

To Add Specific File to Staging Area

Example:
git add index.html

--------------------------------------------------

8. Use "git add ."

To Add All Files to Staging Area

--------------------------------------------------

9. Use "git commit -m "Your Message""

To Save Changes Permanently

Example:
git commit -m "Initial Commit"

--------------------------------------------------

10. Use "git log"

To See Commit History

--------------------------------------------------

11. Use "git log --oneline"

To See Short Commit History

--------------------------------------------------

12. Use "git diff"

To See What Changes Were Made

--------------------------------------------------

13. Use "git branch"

To Check All Branches

--------------------------------------------------

14. Use "git branch branch_name"

To Create New Branch

Example:
git branch feature-login

--------------------------------------------------

15. Use "git checkout branch_name"

To Switch Branch

--------------------------------------------------

16. Use "git checkout -b branch_name"

To Create AND Switch Branch in One Command

--------------------------------------------------

17. Use "git merge branch_name"

To Merge Branch into Current Branch

Example:
git merge feature-login

--------------------------------------------------

18. Use "git clone repository_url"

To Download Repository From GitHub

git clone https://github.com/user/repo.git

19. Use "git remote -v"

To Check Connected Remote Repository

--------------------------------------------------

20. Use "git remote add origin repository_url"
